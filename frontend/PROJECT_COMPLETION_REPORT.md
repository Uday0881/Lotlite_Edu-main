## 💡 Key Learnings & Best Practices

### Design Patterns
1. **Memoization** — Always memoize components with stable props
2. **useCallback** — Wrap event handlers to prevent re-renders
3. **GPU Acceleration** — Use transform & opacity only
4. **Easing Curves** — Use `[0.16, 1, 0.3, 1]` for premium feel

### Accessibility
1. **Focus Indicators** — Always provide visible focus (3px minimum)
2. **Keyboard Navigation** — All interactive elements must be keyboard accessible
3. **ARIA Attributes** — Use aria-expanded, aria-controls, aria-label
4. **Color Contrast** — Target 7:1 ratio for AAA compliance

### Performance
1. **Reduce Motion** — Always respect prefers-reduced-motion
2. **Bundle Size** — Every KB counts; analyze before adding
3. **Animations** — Limit to 240-300ms for micro-interactions
4. **Hover Effects** — Keep subtle; avoid overanimation
